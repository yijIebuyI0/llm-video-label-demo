# LLM辅助短视频种草内容结构化打标Demo

## 项目背景

在短视频种草复盘中，原始数据通常包括播放、互动、收藏、转化等指标，但视频内容特征往往是非结构化的，例如视频标题、内容摘要、评论反馈等。  
本项目尝试使用大语言模型对视频文本信息进行结构化打标，将非结构化内容转化为可分析字段，辅助后续内容表现分析。

## 项目目标

- 使用 Python 读取短视频样本数据
- 基于视频标题、内容摘要、评论摘要构造结构化 Prompt
- 调用大语言模型 API 输出 JSON 格式标签
- 将 LLM 标签结果写回 Excel
- 与人工标签进行一致性对比
- 验证 LLM 辅助内容打标的可行性

## 输入字段

- video_id
- 视频标题
- 视频内容摘要
- 评论摘要
- 人工_视频类型
- 人工_商品露出形式
- 人工_达人类型

## 输出字段

- LLM_视频类型
- LLM_商品露出形式
- LLM_达人类型
- LLM_是否高潜素材
- LLM_判断理由
- 视频类型_一致性
- 商品露出形式_一致性
- 达人类型_一致性

## 技术栈

- Python
- pandas
- openai SDK
- DeepSeek API / OpenAI-compatible API
- JSON结构化输出
- Excel数据处理

## 项目流程

Excel样本数据  
→ Python读取  
→ 构造结构化Prompt  
→ 调用LLM API  
→ 返回JSON标签  
→ 解析并写回Excel  
→ 与人工标签进行一致性对比

## 项目结论

小样本测试表明，LLM可以较好地根据视频标题、内容摘要和评论摘要输出结构化标签，适合用于短视频内容初筛、内容特征提取和人工打标辅助。  
正式业务落地时，可进一步扩展到更大样本，并加入人工抽检、一致性评估和标签体系优化。
